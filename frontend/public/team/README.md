## Photo Naming Convention

Photos should be named using the member's full name in lowercase with hyphens replacing spaces:

**Format:** `{first-name}-{last-name}.jpg`

### Examples:
- Aryan Mundra → `aryan-mundra.jpg`
- Anish Gawande → `anish-gawande.jpg`
- Aditya Chougule → `aditya-chougule.jpg`

## Supported Image Formats

- `.jpg` (recommended)
- `.jpeg`
- `.png`
- `.webp` (for optimized file sizes)

## Photo Requirements

- **Minimum Resolution:** 500x500px (preferably higher for crisp display)
- **File Size:** Keep under 500KB (aim for 100-200KB for web optimization)
- **Format:** Square images work best for the circular display in cards
- **Quality:** High quality headshots or professional photos recommended

## How Photos Are Displayed

1. **Core Team:** Circular profile images in the top hero section
2. **Sabha Members:** Circular profile images in expandable cards
3. **Spotlight Panel:** Large photo overlay when hovering over a member name

## Adding Photos

1. Take/prepare the photo of the team member
2. Save it with the naming convention: `{first-name}-{last-name}.jpg`
3. Place it in the appropriate team folder
4. The website will automatically display the photo

If a photo is missing, the system will fall back to displaying the member's first initial instead.

## Optimization Tips

- Use image compression tools like TinyPNG or ImageOptim
- Consider converting to WebP format for better compression
- Keep aspect ratio close to 1:1 (square) for consistency
- Ensure good lighting and clear visibility of the team member
